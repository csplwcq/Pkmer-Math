傅立叶级数的三角形式

周期为 TTT 的函数 f(x)f(x)f(x) 可以使用傅立叶级数表示为：

f(x)=a02+∑n=1∞(ancos⁡2πnxT+bnsin⁡2πnxT) f(x) = \frac{a_0}{2} + \sum_{n=1}^{\infty} \left( a_n \cos \frac{2\pi n x}{T} + b_n \sin \frac{2\pi n x}{T} \right) f(x)=2a0​​+n=1∑∞​(an​cosT2πnx​+bn​sinT2πnx​)

其中，ana_nan​ 和 bnb_nbn​ 是傅立叶系数，可以通过原函数积分计算得到：

an=2T∫0Tf(x)cos⁡2πnxT dx(n=0,1,2,… ) a_n = \frac{2}{T} \int_{0}^{T} f(x) \cos \frac{2\pi n x}{T} \, dx \quad (n=0,1,2,\dots) an​=T2​∫0T​f(x)cosT2πnx​dx(n=0,1,2,…)

bn=2T∫0Tf(x)sin⁡2πnxT dx(n=0,1,2,… ) b_n = \frac{2}{T} \int_{0}^{T} f(x) \sin \frac{2\pi n x}{T} \, dx \quad (n=0,1,2,\dots) bn​=T2​∫0T​f(x)sinT2πnx​dx(n=0,1,2,…)

周期窗口(积分区间)的选取是任意的，只要长度为一个完整周期即可

  • 定理: 收敛定理, Dirichlet充分条件

三角级数

1. 周期函数

正弦函数是一种常见而简单的周期函数, 例如描述简谐振动的函数

y=Asin⁡(ωt+ϕ) y=A\sin(\omega t+\phi) y=Asin(ωt+ϕ)

实际问题中还有非正弦函数外的周期函数, 如矩形波 周期函数反映了客观世界的周期运动

2. 矩形波展开为正弦函数:

前面介绍过使用函数的幂级数展开函数, 周期函数可以展开为三角函数

f(t)=A0+∑n=1∞Ansin⁡(nωt+ϕn) f(t)=A_{0}+ \sum_{n=1}^{\infty} A_{n}\sin(n\omega t+\phi_{n}) f(t)=A0​+n=1∑∞​An​sin(nωt+ϕn​)

  • A0A_{0}A0​称为直流分量
  • A1sin⁡(ωt+ϕ)A_{1}\sin(\omega t+\phi)A1​sin(ωt+ϕ)称为一次谐波或基波
  • Ansin⁡(ωt+ϕ)A_{n}\sin(\omega t+\phi)An​sin(ωt+ϕ) 称为n次谐波

傅里叶级数

使用三角公式变形级数项

Ansin⁡(nωt+ϕn)=Ansin⁡(ϕ)cos⁡(nωt)+Ancos⁡(ϕn)sin⁡(nωt) A_n\sin(n\omega t+\phi_n)=A_n\sin(\phi)\cos(n\omega t)+A_n\cos(\phi_n)\sin(n\omega t) An​sin(nωt+ϕn​)=An​sin(ϕ)cos(nωt)+An​cos(ϕn​)sin(nωt)

为了方便, 将上式符号改写为

a02+∑n=1∞(ancos⁡(nx)+bnsin⁡(nx)) \dfrac{a_{0}}{2}+\sum_{n=1}^{\infty}(a_{n}\cos(nx)+b_{n}\sin(nx)) 2a0​​+n=1∑∞​(an​cos(nx)+bn​sin(nx))

可以得到，其中ana_nan​ bnb_nbn​均为常数。 这样一来我们把周期为2πω\frac{2\pi}{\omega}ω2π​的三角函数变成了周期为2π2\pi2π的三角级数 下面讨论以T=2πT=2\piT=2π为周期的三角级数

三角函数系

例如: 1, cos(x)cos(x)cos(x), sin⁡(x)\sin(x)sin(x), sin⁡(2x)\sin(2x)sin(2x), cos⁡(2x)\cos(2x)cos(2x)...cos⁡(nx)\cos(nx)cos(nx), sin⁡(nx)\sin(nx)sin(nx) 三角函数系有正交性: 任意两个三角函数系的乘积在区间[−π,π][-\pi,\pi][−π,π]上的定积分=0 证明使用积化和差

周期函数展开为三角函数系

如果周期函数可以展开为

f(x)=a02+∑n=1∞(ancos⁡(nx)+bnsin⁡(nx)) f(x)=\dfrac{a_{0}}{2}+\sum_{n=1}^{\infty}(a_{n}\cos(nx)+b_{n}\sin(nx)) f(x)=2a0​​+n=1∑∞​(an​cos(nx)+bn​sin(nx))

我们不禁要问ana_nan​ bnb_nbn​ 之间存在什么关系，换句话说，如何使用已知信息f(x)f(x)f(x)将ana_nan​ bnb_nbn​表示出来。 为此，我们进一步假设等式两端可以积分。

1. 计算a0a_0a0​

将傅立叶级数的表达式在 [−π,π][- \pi, \pi][−π,π] 区间上积分：

∫−ππf(x) dx=∫−ππ(a02+∑n=1∞(ancos⁡(nx)+bnsin⁡(nx))) dx \int_{-\pi}^{\pi} f(x)\,dx = \int_{-\pi}^{\pi} \left(\dfrac{a_{0}}{2}+\sum_{n=1}^{\infty}(a_{n}\cos(nx)+b_{n}\sin(nx))\right)\,dx ∫−ππ​f(x)dx=∫−ππ​(2a0​​+n=1∑∞​(an​cos(nx)+bn​sin(nx)))dx

由于余弦函数和正弦函数在 [−π,π][- \pi, \pi][−π,π] 内的积分为零，即：

∫−ππcos⁡(nx) dx=0(对于 n≠0) \int_{-\pi}^{\pi} \cos(nx)\,dx = 0 \quad \text{(对于 $n \neq 0$)} ∫−ππ​cos(nx)dx=0(对于 n=0)

∫−ππsin⁡(nx) dx=0(对于所有 n) \int_{-\pi}^{\pi} \sin(nx)\,dx = 0 \quad \text{(对于所有 $n$)} ∫−ππ​sin(nx)dx=0(对于所有 n)

所以，上述积分式简化为：

∫−ππf(x) dx=∫−ππa02 dx=a02⋅2π=a0⋅π \int_{-\pi}^{\pi} f(x)\,dx = \int_{-\pi}^{\pi} \dfrac{a_{0}}{2}\,dx = \dfrac{a_{0}}{2} \cdot 2\pi = a_0 \cdot \pi ∫−ππ​f(x)dx=∫−ππ​2a0​​dx=2a0​​⋅2π=a0​⋅π

因此，

a0=1π∫−ππf(x) dx a_0 = \frac{1}{\pi}\int_{-\pi}^{\pi} f(x)\,dx a0​=π1​∫−ππ​f(x)dx

2. 计算 ana_nan​

为了得到 ana_nan​，将原级数表达式乘以 cos⁡(mx)\cos(mx)cos(mx) （其中 mmm 是任意正整数）后在 [−π,π][- \pi, \pi][−π,π] 区间上积分：

∫−ππf(x)cos⁡(mx) dx=∫−ππ(a02cos⁡(mx)+∑n=1∞(ancos⁡(nx)cos⁡(mx)+bnsin⁡(nx)cos⁡(mx)))dx \int_{-\pi}^{\pi} f(x)\cos(mx)\,dx = \int_{-\pi}^{\pi} \left(\dfrac{a_{0}}{2}\cos(mx) + \sum_{n=1}^{\infty}(a_{n}\cos(nx)\cos(mx) + b_{n}\sin(nx)\cos(mx))\right) dx ∫−ππ​f(x)cos(mx)dx=∫−ππ​(2a0​​cos(mx)+n=1∑∞​(an​cos(nx)cos(mx)+bn​sin(nx)cos(mx)))dx

根据三角函数的正交性：

∫−ππcos⁡(nx)cos⁡(mx) dx={π,当 n=m0,当 n≠m \int_{-\pi}^{\pi} \cos(nx)\cos(mx)\,dx = \begin{cases} \pi, & \text{当 } n = m \\ 0, & \text{当 } n \neq m \end{cases} ∫−ππ​cos(nx)cos(mx)dx={π,0,​当 n=m当 n=m​

∫−ππsin⁡(nx)cos⁡(mx) dx=0(对于所有 n 和 m) \int_{-\pi}^{\pi} \sin(nx)\cos(mx)\,dx = 0 \quad \text{(对于所有 $n$ 和 $m$)} ∫−ππ​sin(nx)cos(mx)dx=0(对于所有 n 和 m)

由此可得：

∫−ππf(x)cos⁡(mx) dx=am⋅π \int_{-\pi}^{\pi} f(x)\cos(mx)\,dx = a_m \cdot \pi ∫−ππ​f(x)cos(mx)dx=am​⋅π

所以：

am=1π∫−ππf(x)cos⁡(mx) dx(其中 m=1,2,3,… ) a_{m}=\dfrac{1}{\pi}\int_{-\pi}^{\pi}f(x)\cos(mx) \, dx \quad \text{(其中 $m = 1, 2, 3, \dots$)} am​=π1​∫−ππ​f(x)cos(mx)dx(其中 m=1,2,3,…)

3. 计算 bnb_nbn​

类似地，为了得到 bnb_nbn​，将原级数表达式乘以 sin⁡(mx)\sin(mx)sin(mx) 后在 [−π,π][- \pi, \pi][−π,π] 区间上积分：

∫−ππf(x)sin⁡(mx) dx=∫−ππ(a02sin⁡(mx)+∑n=1∞(ancos⁡(nx)sin⁡(mx)+bnsin⁡(nx)sin⁡(mx)))dx \int_{-\pi}^{\pi} f(x)\sin(mx)\,dx = \int_{-\pi}^{\pi} \left(\dfrac{a_{0}}{2}\sin(mx) + \sum_{n=1}^{\infty}(a_{n}\cos(nx)\sin(mx) + b_{n}\sin(nx)\sin(mx))\right) dx ∫−ππ​f(x)sin(mx)dx=∫−ππ​(2a0​​sin(mx)+n=1∑∞​(an​cos(nx)sin(mx)+bn​sin(nx)sin(mx)))dx

根据三角函数的正交性：

∫−ππsin⁡(nx)sin⁡(mx) dx={π,当 n=m0,当 n≠m \int_{-\pi}^{\pi} \sin(nx)\sin(mx)\,dx = \begin{cases} \pi, & \text{当 } n = m \\ 0, & \text{当 } n \neq m \end{cases} ∫−ππ​sin(nx)sin(mx)dx={π,0,​当 n=m当 n=m​

∫−ππcos⁡(nx)sin⁡(mx) dx=0(对于所有 n 和 m) \int_{-\pi}^{\pi} \cos(nx)\sin(mx)\,dx = 0 \quad \text{(对于所有 $n$ 和 $m$)} ∫−ππ​cos(nx)sin(mx)dx=0(对于所有 n 和 m)

由此可得：

∫−ππf(x)sin⁡(mx) dx=bm⋅π \int_{-\pi}^{\pi} f(x)\sin(mx)\,dx = b_m \cdot \pi ∫−ππ​f(x)sin(mx)dx=bm​⋅π

所以：

bm=1π∫−ππf(x)sin⁡(mx) dx(其中 m=1,2,3,… ) b_{m}=\dfrac{1}{\pi}\int_{-\pi}^{\pi}f(x)\sin(mx) \, dx \quad \text{(其中 $m = 1, 2, 3, \dots$)} bm​=π1​∫−ππ​f(x)sin(mx)dx(其中 m=1,2,3,…)

于是我们得到了傅立叶级数展开中的所有系数：

a0=12π∫−ππf(x) dx a_0=\frac{1}{2\pi}\int_{-\pi}^{\pi} f(x)\,dx a0​=2π1​∫−ππ​f(x)dx

an=1π∫−ππf(x)cos⁡(nx) dx,(n=1,2,… ) a_{n}=\dfrac{1}{\pi}\int_{-\pi}^{\pi}f(x)\cos(nx) \, dx , (n=1,2,\dots) an​=π1​∫−ππ​f(x)cos(nx)dx,(n=1,2,…)

bn=1π∫−ππf(x)sin⁡(nx) dx,(n=1,2,… ) b_{n}=\dfrac{1}{\pi}\int_{-\pi}^{\pi}f(x)\sin(nx) \, dx , (n=1,2,\dots) bn​=π1​∫−ππ​f(x)sin(nx)dx,(n=1,2,…)

根据假设的前提 T=2π/ωT=2\pi/\omegaT=2π/ω，可以将上式推广至任意周期函数

Dirichlet充分条件

一个定义在[−∞,∞][-\infty,\infty][−∞,∞]的周期为2π2\pi2π的函数f(x)f(x)f(x)，如果在一个周期上可积，那么一定可以做出傅立叶级数。然而f(x)f(x)f(x)的傅立叶级数是否一定收敛？如果它收敛，那它是否一定收敛于f(x)f(x)f(x)？一般来说，这两个答案都不是肯定的，那么 f(x)f(x)f(x) 在怎样的的条件下，它的傅立叶级数不仅收敛，而且收敛于 f(x)f(x)f(x) ？也就是说在满足什么条件下可以展开为傅立叶级数？

Info

设f(x)f(x)f(x)是周期为2π2\pi2π的周期函数，如果满足

  • 在一个周期内连续或只有有限个[[第一类间断点]]
  • 在一个周期内至多只有有限个极值点 那么。的傅立叶级数收敛，并且
  • 当xxx是f(x)f(x)f(x)的连续点时，级数收敛于 f(x)f(x)f(x)
  • 当xxx是f(x)f(x)f(x)的间断点时，级数收敛于12[f(x−)+f(x+)]\frac{1}{2}[f(x^-)+f(x^+)]21​[f(x−)+f(x+)]

复指数形式

下面借由欧拉公式eix=cos⁡x+isin⁡xe^{ix}=\cos x+i\sin xeix=cosx+isinx 将傅里叶级数系数简化成复数指数形式。

f(x)=a02+∑n=1∞(ancos⁡2πnxT+bnsin⁡2πnxT) f(x) = \frac{a_0}{2} + \sum_{n=1}^{\infty} \left( a_n \cos \frac{2\pi n x}{T} + b_n \sin \frac{2\pi n x}{T} \right) f(x)=2a0​​+n=1∑∞​(an​cosT2πnx​+bn​sinT2πnx​)

使用eix=cos⁡x+isin⁡xe^{ix}=\cos x+i\sin xeix=cosx+isinx 将正弦和余弦表示为复指数形式

cos⁡2πnxT=ei2πnxT+e−i2πnxT2 \cos \frac{2\pi n x}{T} = \frac{e^{i \frac{2\pi n x}{T}} + e^{-i \frac{2\pi n x}{T}}}{2} cosT2πnx​=2eiT2πnx​+e−iT2πnx​​

sin⁡2πnxT=ei2πnxT−e−i2πnxT2i \sin \frac{2\pi n x}{T} = \frac{e^{i \frac{2\pi n x}{T}} - e^{-i \frac{2\pi n x}{T}}}{2i} sinT2πnx​=2ieiT2πnx​−e−iT2πnx​​

代入傅立叶级数得到：

f(x)=a02+∑n=1∞[an(ei2πnxT+e−i2πnxT2)+bn(ei2πnxT−e−i2πnxT2i)] f(x) = \frac{a_0}{2} + \sum_{n=1}^{\infty} \left[ a_n \left( \frac{e^{i \frac{2\pi n x}{T}} + e^{-i \frac{2\pi n x}{T}}}{2} \right) + b_n \left( \frac{e^{i \frac{2\pi n x}{T}} - e^{-i \frac{2\pi n x}{T}}}{2i} \right) \right] f(x)=2a0​​+n=1∑∞​[an​(2eiT2πnx​+e−iT2πnx​​)+bn​(2ieiT2πnx​−e−iT2πnx​​)]

重新整理，我们可以将上面的表达式转化为：

f(x)=∑n=−∞∞cnei2πnxT f(x) = \sum_{n=-\infty}^{\infty} c_n e^{i \frac{2\pi n x}{T}} f(x)=n=−∞∑∞​cn​eiT2πnx​

其中，复数系数 cnc_ncn​ 定义为：

cn=1T∫0Tf(x)e−i2πnxT dx c_n = \frac{1}{T} \int_{0}^{T} f(x) e^{-i \frac{2\pi n x}{T}} \, dx cn​=T1​∫0T​f(x)e−iT2πnx​dx

复数系数与原傅立叶系数的关系

对于 n>0n > 0n>0：

cn=an−ibn2 c_n = \frac{a_n - i b_n}{2} cn​=2an​−ibn​​

对于 n<0n < 0n<0：

cn=a∣n∣+ib∣n∣2 c_n = \frac{a_{|n|} + i b_{|n|}}{2} cn​=2a∣n∣​+ib∣n∣​​

对于 n=0n = 0n=0：

c0=a02 c_0 = \frac{a_0}{2} c0​=2a0​​

这样，原来的傅立叶级数形式就可以简化为复数指数形式。

总结

利用欧拉公式可以将傅立叶级数的三角形式转换为复数指数形式。这种形式在许多应用中更为方便，尤其在涉及到对称性或频谱分析时。傅立叶系数 cnc_ncn​ 可以通过直接对函数 f(x)f(x)f(x) 进行复数指数形式的积分来计算，并且与原三角形式中的 ana_nan​ 和 bnb_nbn​ 有直接的关系。
