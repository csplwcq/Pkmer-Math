质心

质心（或称为形心）是一个物体的质量分布的“平均位置”。对于形状规则、密度均匀的物体，质心通常位于物体的几何中心。然而，对于形状不规则或密度不均匀的物体，质心的计算需要通过重积分来完成。

1. 质心的定义

假设一个三维物体VVV的质量密度函数为ρ(x,y,z)\rho(x, y, z)ρ(x,y,z)，质心(xˉ,yˉ,zˉ)(\bar{x}, \bar{y}, \bar{z})(xˉ,yˉ​,zˉ)的坐标可以通过以下公式计算：

xˉ=1M∭Vxρ(x,y,z) dV\bar{x} = \frac{1}{M} \iiint_V x \rho(x, y, z) \, dV xˉ=M1​∭V​xρ(x,y,z)dV

yˉ=1M∭Vyρ(x,y,z) dV\bar{y} = \frac{1}{M} \iiint_V y \rho(x, y, z) \, dV yˉ​=M1​∭V​yρ(x,y,z)dV

zˉ=1M∭Vzρ(x,y,z) dV\bar{z} = \frac{1}{M} \iiint_V z \rho(x, y, z) \, dV zˉ=M1​∭V​zρ(x,y,z)dV

其中MMM为物体的总质量，可以通过以下积分计算：

M=∭Vρ(x,y,z) dVM = \iiint_V \rho(x, y, z) \, dV M=∭V​ρ(x,y,z)dV

2. 质心的计算步骤

1. 确定质量密度函数ρ(x,y,z)\rho(x, y, z)ρ(x,y,z)

如果物体的密度是均匀的，即ρ(x,y,z)=ρ0\rho(x, y, z) = \rho_0ρ(x,y,z)=ρ0​为常数，则密度可以从积分中提出来，这样计算会简化很多。

2. 设定积分区域VVV

根据物体的形状，确定积分区域VVV，这可能是某个简单几何形状（如立方体、球、圆柱等）或通过一定参数化的复杂形状。

3. 计算总质量MMM

使用总质量公式M=∭Vρ(x,y,z) dVM = \iiint_V \rho(x, y, z) \, dVM=∭V​ρ(x,y,z)dV来计算物体的总质量。这一步可以在计算质心坐标之前或者在计算质心坐标的过程中完成。

4. 计算质心坐标

分别计算xˉ\bar{x}xˉ、yˉ\bar{y}yˉ​和zˉ\bar{z}zˉ的积分。这些积分表示质量在各个方向上的加权平均。

3. 示例

考虑一个密度均匀的半球（上半部分），其半径为RRR，密度为ρ0\rho_0ρ0​。我们计算其质心的坐标。

1. 确定密度函数

因为密度均匀，ρ(x,y,z)=ρ0\rho(x, y, z) = \rho_0ρ(x,y,z)=ρ0​。

2. 设定积分区域

我们使用球坐标系来进行积分，设zzz轴垂直于半球平面，原点位于球心处，则积分区域VVV由如下参数描述：

0≤r≤R,0≤θ≤π,0≤ϕ≤π0 \leq r \leq R, \quad 0 \leq \theta \leq \pi, \quad 0 \leq \phi \leq \pi 0≤r≤R,0≤θ≤π,0≤ϕ≤π

3. 计算总质量MMM

总质量为：

M=ρ0∭VdV=ρ0∫0π∫0π∫0Rr2sin⁡θ dr dθ dϕM = \rho_0 \iiint_V dV = \rho_0 \int_0^\pi \int_0^\pi \int_0^R r^2 \sin\theta \, dr \, d\theta \, d\phi M=ρ0​∭V​dV=ρ0​∫0π​∫0π​∫0R​r2sinθdrdθdϕ

通过计算，得到：

M=23πR3ρ0M = \frac{2}{3} \pi R^3 \rho_0 M=32​πR3ρ0​

4. 计算质心坐标

对于xˉ\bar{x}xˉ和yˉ\bar{y}yˉ​：

由于形状的对称性，可以直接得到xˉ=0\bar{x} = 0xˉ=0，yˉ=0\bar{y} = 0yˉ​=0。

对于zˉ\bar{z}zˉ：

zˉ=1M∭Vzρ(x,y,z) dV=1Mρ0∫0π∫0π∫0Rrcos⁡θ⋅r2sin⁡θ dr dθ dϕ\bar{z} = \frac{1}{M} \iiint_V z \rho(x, y, z) \, dV = \frac{1}{M} \rho_0 \int_0^\pi \int_0^\pi \int_0^R r \cos\theta \cdot r^2 \sin\theta \, dr \, d\theta \, d\phi zˉ=M1​∭V​zρ(x,y,z)dV=M1​ρ0​∫0π​∫0π​∫0R​rcosθ⋅r2sinθdrdθdϕ

简化并积分后得到：

zˉ=3R8\bar{z} = \frac{3R}{8} zˉ=83R​

所以，半球的质心位于(0,0,3R8)(0, 0, \frac{3R}{8})(0,0,83R​)。

通过这种方法，我们可以计算出任意复杂形状和密度分布物体的质心位置。

能否将三个坐标看作一个位矢，进而将积分简化为向量积分

是的，你可以将质心的三个坐标xˉ\bar{x}xˉ、yˉ\bar{y}yˉ​、zˉ\bar{z}zˉ看作一个位矢r=(x,y,z)\mathbf{r} = (x, y, z)r=(x,y,z)的积分，并将质心计算简化为一个向量积分。这种方法不仅简化了表达式，而且能够更直观地理解质心的物理意义。

向量形式的质心计算

假设一个物体的质量密度函数为ρ(x,y,z)\rho(x, y, z)ρ(x,y,z)，质心rˉ\mathbf{\bar{r}}rˉ的向量表达式为：

rˉ=1M∭Vrρ(x,y,z) dV\mathbf{\bar{r}} = \frac{1}{M} \iiint_V \mathbf{r} \rho(x, y, z) \, dV rˉ=M1​∭V​rρ(x,y,z)dV

其中：

  • rˉ=(xˉ,yˉ,zˉ)\mathbf{\bar{r}} = (\bar{x}, \bar{y}, \bar{z})rˉ=(xˉ,yˉ​,zˉ)是质心的位置向量。
  • r=(x,y,z)\mathbf{r} = (x, y, z)r=(x,y,z)是位置向量。
  • M=∭Vρ(x,y,z) dVM = \iiint_V \rho(x, y, z) \, dVM=∭V​ρ(x,y,z)dV是物体的总质量。

解释

向量rˉ\mathbf{\bar{r}}rˉ表示物体质心的位置。通过将三个质心坐标看作一个整体向量r=(x,y,z)\mathbf{r} = (x, y, z)r=(x,y,z)进行积分，可以将质心的计算简化为一个向量积分。

计算步骤

  1. 计算总质量MMM：计算物体的总质量MMM，方法与之前相同。

    M=∭Vρ(x,y,z) dVM = \iiint_V \rho(x, y, z) \, dV M=∭V​ρ(x,y,z)dV

  2. 计算质心向量rˉ\mathbf{\bar{r}}rˉ：使用向量积分形式计算质心的位置向量rˉ\mathbf{\bar{r}}rˉ：

    rˉ=1M∭Vrρ(x,y,z) dV\mathbf{\bar{r}} = \frac{1}{M} \iiint_V \mathbf{r} \rho(x, y, z) \, dV rˉ=M1​∭V​rρ(x,y,z)dV

    这里的积分实际上是对每个坐标xxx、yyy、zzz分别进行积分，但由于采用向量形式，表达式更加简洁。

示例

考虑之前的半球例子，将质心计算转化为向量积分：

  1. 总质量：

    M=ρ0∭VdV=ρ023πR3M = \rho_0 \iiint_V dV = \rho_0 \frac{2}{3} \pi R^3 M=ρ0​∭V​dV=ρ0​32​πR3

  2. 质心向量：

    rˉ=1Mρ0∭V(x,y,z) dV\mathbf{\bar{r}} = \frac{1}{M} \rho_0 \iiint_V (x, y, z) \, dV rˉ=M1​ρ0​∭V​(x,y,z)dV

    由于xxx和yyy方向对称，xˉ=0\bar{x} = 0xˉ=0，yˉ=0\bar{y} = 0yˉ​=0。我们只需计算zzz分量：

    zˉ=1Mρ0∭Vz dV\bar{z} = \frac{1}{M} \rho_0 \iiint_V z \, dV zˉ=M1​ρ0​∭V​zdV

    最终，得到的质心位置向量为：

    rˉ=(0,0,3R8)\mathbf{\bar{r}} = \left(0, 0, \frac{3R}{8}\right) rˉ=(0,0,83R​)

总结

通过将质心坐标看作一个向量r=(x,y,z)\mathbf{r} = (x, y, z)r=(x,y,z)，我们可以使用向量积分的形式来简化质心的计算。这种方法统一了各坐标的计算步骤，使得质心计算更为简洁和直观。
