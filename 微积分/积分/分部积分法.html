假设 h(x)h(x)h(x) 与 k(x)k(x)k(x) 是两个连续可导函数。由乘积法则可知

d(hk)dx=dhdxk+hdkdx \frac{d(hk)}{dx} = \frac{dh}{dx}k + h\frac{dk}{dx} dxd(hk)​=dxdh​k+hdxdk​

对上述等式两边求不定积分，得

hk=∫(dhdxk+hdkdx)dx=∫hdk+∫kdh hk = \int \left(\frac{dh}{dx}k + h\frac{dk}{dx}\right)dx = \int hdk + \int kdh hk=∫(dxdh​k+hdxdk​)dx=∫hdk+∫kdh

移项整理，得不定积分形式的分部积分方程

∫dhdxkdx=hk−∫hdkdxdx \int \frac{dh}{dx}kdx = hk - \int h\frac{dk}{dx}dx ∫dxdh​kdx=hk−∫hdxdk​dx

由以上等式我们可以推导出分部积分法在区间[a,A][a, A][a,A]的定积分形式

定积分形式

∫aAdhdxkdx=[hk]aA−∫aAhdkdxdx \int_{a}^{A}\frac{dh}{dx}kdx = \left[hk\right]_{a}^{A} - \int_{a}^{A}h\frac{dk}{dx}dx ∫aA​dxdh​kdx=[hk]aA​−∫aA​hdxdk​dx

已经积出的部分 [hk]aA\left[hk\right]_{a}^{A}[hk]aA​ 可以代入上下限$$a, A$$表示为以下等式，

[hk]aA=h(A)k(A)−h(a)k(a) \left[hk\right]_{a}^{A} = h(A)k(A) - h(a)k(a) [hk]aA​=h(A)k(A)−h(a)k(a)

而以上这条等式可以通过函数求导乘积法则，以及微积分基本定理通过以下方式倒推并得以验证

h(A)k(A)−h(a)k(a)=∫aAd(hk)dxdx=∫aA(dhdxk+hdkdx)dx=∫aAkdh+∫aAhdk h(A)k(A) - h(a)k(a) = \int_{a}^{A}\frac{d(hk)}{dx}dx = \int_{a}^{A}\left(\frac{dh}{dx}k + h\frac{dk}{dx}\right)dx = \int_{a}^{A}kdh + \int_{a}^{A}hdk h(A)k(A)−h(a)k(a)=∫aA​dxd(hk)​dx=∫aA​(dxdh​k+hdxdk​)dx=∫aA​kdh+∫aA​hdk

在传统的微积分教材里分部积分法通常写成不定积分形式：

∫f(x)g′(x)dx=f(x)g(x)−∫f′(x)g(x)dx \int f(x)g'(x)dx = f(x)g(x) - \int f'(x)g(x)dx ∫f(x)g′(x)dx=f(x)g(x)−∫f′(x)g(x)dx

如果更简单些，令 u=f(x)u = f(x)u=f(x)、v=g(x)v = g(x)v=g(x)，微分 du=f′(x)dxdu = f'(x)dxdu=f′(x)dx 和 dv=g′(x)dxdv = g'(x)dxdv=g′(x)dx，就可以得到更常见到的形式：

∫u dv=uv−∫v du \int u\,dv = uv - \int v\,du ∫udv=uv−∫vdu

注意，上面的原式中含有 ggg 的导数；在使用这个规则时必须先找到不定积分 ggg，并且积分 ∫gf′dx\int gf'dx∫gf′dx 必须是可积的

在级数的离散分析中也可以用到类似的公式表达，称为分部求和。 另一可用的表达方式可以将原表达方式里的因子仅写成 fff 和 ggg，但缺点是引进了镶套积分：

∫fg dx=f∫g dx−∫(f′∫g dx) dx \int fg\,dx = f\int g\,dx - \int \left(f' \int g\,dx\right)\,dx ∫fgdx=f∫gdx−∫(f′∫gdx)dx

这个表达方式只有当 fff 是连续可导而且 ggg 是连续的时才有效。

在黎曼-斯蒂尔吉斯积分和勒贝格-斯蒂尔吉斯积分有更多分部积分的公式。

提示：部分积分下面这样更复杂一点的积分运算里也是有效的：

∫uv dw=uvw−∫uw dv−∫vw du \int u v\,dw = uvw - \int uw\,dv - \int vw\,du ∫uvdw=uvw−∫uwdv−∫vwdu

选择 uuu 和 dvdvdv 的规则（ILATE 规则）

在选择 uuu 和 dvdvdv 时，一种常用的记忆方法是“ILATE”规则，这是各类型函数的优先级：

  • I (Inverse trigonometric functions) - 反三角函数：arcsin⁡x\arcsin xarcsinx, arctan⁡x\arctan xarctanx, 等。
  • L (Logarithmic functions) - 对数函数：ln⁡x\ln xlnx, log⁡bx\log_b xlogb​x。
  • A (Algebraic functions) - 代数函数：x2x^2x2, 3x+73x + 73x+7 等。
  • T (Trigonometric functions) - 三角函数：sin⁡x\sin xsinx, cos⁡x\cos xcosx。
  • E (Exponential functions) - 指数函数：exe^xex, 2x2^x2x。

应用ILATE规则

  1. 在多个函数的乘积中，尽可能按照ILATE的顺序来选择 uuu。
  2. 选择 uuu 后，剩余部分作为 dvdvdv。

这个顺序基于将 uuu 设置为积分后会简化的函数的想法。对数和反三角函数在微分后简化为代数形式，而代数函数和三角函数微分后一般不会增加复杂度，指数函数微分后形式不变。
